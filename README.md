### Чем я занимаюсь  
- 💼 **Разработка отчетности:** Создаю дашборды любой сложности от финмоделей и операционной аналитики до дашбордов с производственными near real-time процессами и технологиями нефтедобывающей промышленности.
- ➡️ **Миграция данных:** Переношу отчетность из зарубежной системы (Power BI) на российскую платформу (Visiology).
